# Industrial Hub

Sistema interno de operações industriais.

## Módulos
- OEE + Disponibilidade de Máquinas
- QMS — Qualidade e Não Conformidades
- Melhoria Contínua + Engenharia
- Portal do Colaborador

## Stack
- Frontend: Angular 17+ + Tailwind CSS
- Backend: Java 21 + Spring Boot 3
- Banco: PostgreSQL 16
- Infra: Docker + docker-compose
