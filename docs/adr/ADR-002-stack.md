## ADR-002: Stack Tecnológico
**Status**: Aprovado
**Data**: 2026-05-09

### Contexto
Desenvolvedor com experiência em Java e Angular. Projeto interno sem custo definido.

### Decisão
Angular 17+ + Tailwind CSS no frontend. Java 21 + Spring Boot 3 no backend.

### Consequências
- Stack familiar reduz curva de aprendizado
- Spring Boot 3 com Java 21 LTS garante suporte longo
